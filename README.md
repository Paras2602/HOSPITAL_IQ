# HospitalIQ

HospitalIQ is an advanced Healthcare Management System that integrates AI-powered clinical diagnosis, automated prescription generation, and end-to-end patient care workflows.

## Features
- **AI Symptom Checker**: Multi-disease diagnostic engine with high-confidence predictions.
- **Clinical Dashboard**: Real-time analytics and patient management for doctors and administrators.
- **Automated Prescriptions**: LLM-driven medication and precaution guidelines with PDF generation.
- **Role-Based Access**: Secure portals for Patients, Doctors, Lab Technicians, and Admins.
- **Audit Logging**: Comprehensive activity tracking for clinical and system events.

## Tech Stack
- **Frontend**: Next.js, TypeScript, TailwindCSS, Lucide Icons.
- **Backend**: FastAPI (Python), SQLAlchemy (SQLite/PostgreSQL).
- **AI/ML**: Scikit-learn models for disease prediction.

## Setup
1. Clone the repository.
2. Run `start_backend.bat` to initialize the API.
3. Run `start_frontend.bat` to launch the dashboard.
